# SQL

## 🐣 SQL 이 뭔가요?
:::tip
Structured Query Language 구조화 쿼리(질의) 언어  
RDBMS에서 데이터를 관리하는 언어 (데이터베이스용 프로그래밍 언어)
1. (DDL) 데이터 정의 ex. 데이터베이스 스키마 관리
2. (DML) 데이터 조작 ex. 데이터 CRUD
3. (DCL) 데이터 제어 ex. 객체 권한, DB 접근 권한, 트랜잭션 명시 등
4. (TCL) 트랜잭션 제어
:::
